Welcome to the XSB for Mac OS X Installer

XSB is a Logic Programming and Deductive Database system for Unix and Windows. It is being developed at the Computer Science Department of the Stony Brook University, in collaboration with Katholieke Universiteit Leuven, Universidade Nova de Lisboa, Uppsala Universitet and XSB, Inc.

This installer guides you through the steps necessary to install XSB for Mac OS X. XSB is installed on the /opt/local/share directory. A symbolic link to the xsb executable is created on /opt/local/bin, which should be in your path. A link to the XSB installation folder is automatically created on the Applications folder. The installer sets the XSB_DIR environment variable pointing to the XSB installation directory. You may need to logout and login again to activate the environment variable.

To get started, click Continue.
