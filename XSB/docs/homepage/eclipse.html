• The Eclipse Plugin for Prolog Homepage

    • This link provides information about a basic Eclipse Plugin for Prolog. Using the plugin, the Eclipse workspace gives access to the Prolog command-line interface, and the editor gives syntactic highlighting for Prolog. The plugin works fine for XSB and has allowed some users to use Eclipse as their XSB development environment. It was apparently written for Ciao, so their logo appears in several places, but there seems to be nothing Ciao-specific in the plugin.

    We do not have the resources to actively maintain or extend the plugin ourselves, but welcome support for the Eclipse Prolog environment by the community.
